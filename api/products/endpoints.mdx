## Create product

<Note>
Requires `ADMIN` role
</Note>

Create a new product in the catalog.

```http
POST /products
```

### Request body

<ParamField body="name" type="string" required>
  Product name (max 150 characters)
</ParamField>

<ParamField body="description" type="string">
  Product description (max 1000 characters)
</ParamField>

<ParamField body="brand" type="string" required>
  Brand name (max 100 characters)
</ParamField>

<ParamField body="categoryId" type="integer" required>
  ID of the category this product belongs to. Must be a positive integer.
</ParamField>

<ParamField body="price" type="number" required>
  Product price. Must be a non-negative decimal value with up to 2 decimal places.
</ParamField>

<ParamField body="currency" type="string" required>
  ISO 3-letter uppercase currency code (e.g., USD, EUR, GBP)
</ParamField>

<ParamField body="status" type="string" required>
  Product status. Valid values: `ACTIVE`, `INACTIVE`, `OUT_OF_STOCK`, `DISCONTINUED`
</ParamField>

<CodeGroup>
```bash cURL
curl -X POST "http://localhost:8083/products" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Premium Wireless Headphones",
    "description": "High-quality noise-canceling headphones with 30-hour battery life",
    "brand": "AudioTech",
    "categoryId": 5,
    "price": 199.99,
    "currency": "USD",
    "status": "ACTIVE"
  }'
```

```javascript JavaScript
const response = await fetch('http://localhost:8083/products', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${token}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'Premium Wireless Headphones',
    description: 'High-quality noise-canceling headphones with 30-hour battery life',
    brand: 'AudioTech',
    categoryId: 5,
    price: 199.99,
    currency: 'USD',
    status: 'ACTIVE'
  })
});

const product = await response.json();
```

```python Python
import requests

response = requests.post(
    'http://localhost:8083/products',
    headers={
        'Authorization': f'Bearer {token}',
        'Content-Type': 'application/json'
    },
    json={
        'name': 'Premium Wireless Headphones',
        'description': 'High-quality noise-canceling headphones with 30-hour battery life',
        'brand': 'AudioTech',
        'categoryId': 5,
        'price': 199.99,
        'currency': 'USD',
        'status': 'ACTIVE'
    }
)

product = response.json()
```
</CodeGroup>

## List products

Retrieve a paginated list of products with optional filters.

```http
GET /products
```

### Query parameters

<ParamField query="categoryId" type="integer">
  Filter products by category ID
</ParamField>

<ParamField query="brand" type="string">
  Filter products by brand name
</ParamField>

<ParamField query="status" type="string">
  Filter by product status. Valid values: `ACTIVE`, `INACTIVE`, `OUT_OF_STOCK`, `DISCONTINUED`
</ParamField>

<ParamField query="page" type="integer" default="0">
  Page number (zero-indexed)
</ParamField>

<ParamField query="size" type="integer" default="10">
  Number of items per page
</ParamField>

<ParamField query="sort" type="string">
  Sort criteria (e.g., `name,asc` or `price,desc`)
</ParamField>

<CodeGroup>
```bash cURL
curl -X GET "http://localhost:8083/products?categoryId=5&status=ACTIVE&page=0&size=10" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

```javascript JavaScript
const params = new URLSearchParams({
  categoryId: '5',
  status: 'ACTIVE',
  page: '0',
  size: '10'
});

const response = await fetch(`http://localhost:8083/products?${params}`, {
  headers: {
    'Authorization': `Bearer ${token}`
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'http://localhost:8083/products',
    headers={'Authorization': f'Bearer {token}'},
    params={
        'categoryId': 5,
        'status': 'ACTIVE',
        'page': 0,
        'size': 10
    }
)

data = response.json()
```
</CodeGroup>

### Response

<ResponseField name="content" type="array">
  Array of product objects
</ResponseField>

<ResponseField name="page" type="integer">
  Current page number (zero-indexed)
</ResponseField>

<ResponseField name="size" type="integer">
  Number of items per page
</ResponseField>

<ResponseField name="totalElements" type="integer">
  Total number of products matching the filters
</ResponseField>

<ResponseField name="totalPages" type="integer">
  Total number of pages
</ResponseField>

<ResponseField name="last" type="boolean">
  Whether this is the last page
</ResponseField>

## Delete product

<Note>
Requires `ADMIN` role
</Note>

Soft delete a product. The product will be marked as deleted but not removed from the database.

```http
DELETE /products/{productId}
```

### Path parameters

<ParamField path="productId" type="integer" required>
  The unique identifier of the product to delete
</ParamField>

<CodeGroup>
```bash cURL
curl -X DELETE "http://localhost:8083/products/123" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

```javascript JavaScript
const response = await fetch('http://localhost:8083/products/123', {
  method: 'DELETE',
  headers: {
    'Authorization': `Bearer ${token}`
  }
});

const result = await response.json();
```

```python Python
import requests

response = requests.delete(
    'http://localhost:8083/products/123',
    headers={'Authorization': f'Bearer {token}'}
)

result = response.json()
```
</CodeGroup>

### Response

<ResponseField name="message" type="string">
  Confirmation message
</ResponseField>

<CodeGroup>
```json 200 OK
{
  "message": "Product deleted successfully"
}
```

```json 404 Not Found
{
  "timestamp": "2026-03-03T12:45:00Z",
  "status": 404,
  "error": "Not Found",
  "message": "Product not found with id: 999",
```

## Search products

Search for products by name or brand using a query string.

```http
GET /products/search
```

### Query parameters

<ParamField query="query" type="string" required>
  Search term to match against product names and brands
</ParamField>

<CodeGroup>
```bash cURL
curl -X GET "http://localhost:8083/products/search?query=wireless" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

```javascript JavaScript
const params = new URLSearchParams({
  query: 'wireless'
});

const response = await fetch(`http://localhost:8083/products/search?${params}`, {
  headers: {
    'Authorization': `Bearer ${token}`
  }
});

const products = await response.json();
```

```python Python
import requests

response = requests.get(
    'http://localhost:8083/products/search',
    headers={'Authorization': f'Bearer {token}'},
    params={'query': 'wireless'}
)

products = response.json()
```
</CodeGroup>
